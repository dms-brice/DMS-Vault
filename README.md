# DMS Vault — Décors des Mers du Sud

Vault Obsidian partagée. Système de gestion des connaissances métier : stratégie, création, outils, prospection, communication.

## Structure

```
SYSTÈME DMS/
├── 00-CONTEXTE (comprendre)          → Cadre stratégique, analyse, fondamentaux
├── 01-MOTEUR DMS (réfléchir)         → Processus, réflexion, méthode
├── 02-OUTILS (produire)              → Ressources, modèles, templates
├── 03-Contenu & Communication (communiquer) → Messaging, contenu, canal
└── 04-Prospection & Vente (vendre)   → Pipeline, cas clients, tactiques
```

Chaque dossier numéroté porte un verbe d'action qui décrit son rôle opérationnel.

## Collaboration

- **Brice** : Stratégie, systèmes, commercial. Rôle : diffuser.
- **Carole** : Direction artistique, création, 30 ans d'expertise. Rôle : créer.

## Synchronisation GitHub

Vault = repository local avec remote GitHub : `https://github.com/dms-brice/DMS-Vault.git`

**Workflow quotidien** :
1. Modifier dans Obsidian
2. Terminal : `git add .`
3. Terminal : `git commit -m "description claire"`
4. Terminal : `git push`

Voir [`/Immerger.md`](Immerger.md) pour détails d'installation. Voir `.obsidian/README` pour config plugins.
