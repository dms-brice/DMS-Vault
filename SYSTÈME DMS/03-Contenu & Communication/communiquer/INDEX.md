# communiquer — Contenu finalisé & Messaging

**Parent** : `[[../INDEX]]` (03-Contenu & Communication)

## Objectif

**Contenu prêt à diffuser**. Tout ce qui sort vers l'audience : copies, articles, posts, emails, descriptions produit.

## Contenu typique

- **arguments/** → Arguments de vente core (par univers / segment)
- **messaging/** → Copies par marché (Suisse / Réunion)
- **contenus/** → Articles, posts réseaux, emails
- **descriptions/** → Fiches produits, descriptions site
- **visuels/** → Moodboards, références, briefs créatifs finalisés

## Convention

- **Argument core** = réutilisable, générique → Prospects vault l'adapte
- **Messaging** = contextualisé par marché (Suisse premium, Réunion relation)
- **Contenu** = finalisé OU en review

## Important

L'émotion précède l'achat. Tout ce qui est ici doit refléter ça :
- Tonalité : exploration, découverte, intimité
- Visuels : univers botanique, gravures XIXe
- Message : désir avant transaction

## Workflow

1. Créer brouillon
2. Valider avec Carole (créativité) & Brice (stratégie)
3. Finalisé → ce dossier
4. Prospects vault l'utilise dans ANGLES
