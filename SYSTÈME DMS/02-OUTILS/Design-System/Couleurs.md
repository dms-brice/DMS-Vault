## Accessibilité

- Contraste minimum requis (WCAG AA) : Tous les textes respectent 4.5:1
- Palette colorblind-friendly : Vérifier palette

## Usage

**À faire** :
- Utiliser la palette DMS uniquement
- Vérifier contraste pour lisibilité

**À éviter** :
- Inventer nouvelles couleurs
- Mélanger avec palettes externes
- Utiliser des dégradés non-approuvés
