# comprendre — Exploration & Documentation stratégique

**Parent** : `[[../INDEX]]` (00-CONTEXTE)

## Objectif

Notes d'exploration libre, analyses contextuelles, découvertes, brouillons stratégiques.

Tout ce qui aide à **comprendre le cadre** sans être formellement documenté ailleurs.

## Contenu typique

- Notes marché (Suisse, Réunion, tendances)
- Analyses audience (comportement achat, émotions)
- Positionnement brainstorm
- Analyses concurrence
- Insight univers (botanique, gravures XIXe, indiennage)
- Questions stratégiques en cours
- Brouillons avant formalisation

## Convention

- **Draft** = libre, non-finalisé
- **Formalisé** = prêt à être référencé par Prospects vault

## Workflow

1. Explorer / prendre notes
2. Quand stabilisé → formaliser dans `[[../]]` parent
3. Quand en production → référencer dans Prospects vault
